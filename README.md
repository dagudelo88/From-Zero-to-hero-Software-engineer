# From Zero to Hero Software Engineer

A comprehensive roadmap and reference guide for modern software engineering, covering everything from computer science fundamentals and paradigms to infrastructure, system architecture, and the emerging AI landscape.

## 📌 Project Overview

This repository serves as a structured curriculum and knowledge base for anyone looking to master software engineering. It is organized into distinct learning modules that build upon each other, taking you from foundational concepts to advanced, modern techniques.

## 🗂️ Map of Content

> For a complete bird's-eye view of all concepts covered, see the master index at [`Docs/software_engineering_concepts.md`](Docs/software_engineering_concepts.md).

The detailed guides are located in the `From Zero To Hero/` directory, categorized as follows:

### 1. Computer Science Fundamentals
*Core concepts that form the basis of all software development.*
- Data Structures
- Algorithms & Analysis

### 2. Programming Paradigms & Architecture
*Ways of structuring code and system design.*
- Object-Oriented Programming (OOP)
- Design Patterns (Creational, Structural, Behavioral, Enterprise)
- Functional Programming
- Architecture Styles

### 3. Core Systems & Infrastructure (Work in Progress/Planned)
*The environment where software runs and data lives.*
- Operating Systems
- Computer Networks
- Databases & Caching

### 4. Web Architecture & APIs
*How systems communicate and stay secure.*
- API Design 
- Authentication & Authorization
- Security Principles

### 5. Frontend Ecosystem
*Building modern, performant, and accessible user interfaces.*
- Markup, Styling, JavaScript/TypeScript
- Frameworks & State Management

### 6. Backend Ecosystem & Distributed Systems
*Building highly available and scalable server-side logic.*
- Backend Frameworks
- Microservices & Messaging
- Distributed Systems Concepts

### 7. Cloud, DevOps & Deployment
*Shipping software securely, efficiently, and observably.*
- Containerization & CI/CD
- Cloud Platforms & Infrastructure as Code (IaC)
- Observability & Monitoring

### 8. Software Engineering Practices
*Methodologies and quality standards for reliable software delivery.*
- Coding Principles & Testing Strategies
- Deployment & Process Methodologies

### 9. Machine Learning, AI & Large Language Models (LLM)
*Modern AI engineering, foundational machine learning, and intelligent agentic workflows.*
- ML Fundamentals & Deep Learning
- Modern NLP, LLMs, and RAG
- Agentic Workflows & Fine-Tuning

### 10. Emerging & Specialized Tech
*Niche and future-looking technologies.*
- Web3 & Blockchain
- Mobile & Cross-Platform
- Performance, Rust, WASM

## 📖 How to Navigate

- **Docs:** Start with `Docs/software_engineering_concepts.md` for a comprehensive list of all topics that a modern software engineer should know.
- **Deep Dives:** Browse through the numbered folders in `From Zero To Hero/` to read detailed markdown guides on each topic. The structure follows a logical progression, making it easy to read from start to finish or look up a specific topic.
